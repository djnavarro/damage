# damage

<!-- badges: start -->
<!-- badges: end -->

The goal of damage is to make generative art. You can install the
development version from [GitHub](https://github.com/) with:

``` r
remotes::install_github("djnavarro/damage")
```

To generate pieces:

``` r
damage::damage(seed = 1, filename = "damage_1.png")
```
